# Royal Hospitality 🏨

**Royal Hospitality** is a modern and responsive hotel website built using **HTML**, **CSS**, **JavaScript**, and **Bootstrap 5**.  
The website provides an intuitive experience for visitors to explore hotel services, rooms, dining, and make bookings.

---

## 🚀 Website Pages
- **Home** – Welcome section and overview  
- **Menu** – Food and dining options  
- **Rooms** – Details of available rooms  
- **Halls** – Event and banquet hall information  
- **Booking** – Room and hall booking form  
- **Contact** – Contact details and form  

---

## ✨ Features
- Fully responsive design for desktops, tablets, and mobiles  
- Modern, clean, and user-friendly interface  
- Easy navigation between pages  
- Detailed rooms and halls information  
- Dining menu display  
- Booking form for rooms and halls  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5
