# Royal Hospitality

Royal Hospitality is a full stack Django web application for hotel and restaurant management.

## Features

- Room booking system
- Table reservation system
- Contact form
- Automated email confirmation on booking
- Admin panel for managing bookings
- Static media and templates integration

## Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, JavaScript
- Database: SQLite (Development)
- Email Integration: SMTP configuration

## Installation

1. Clone the repository
2. Create a virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Run migrations:
   python manage.py migrate
5. Start server:
   python manage.py runserver

## Project Structure

- hotel/ → App logic
- royal_hospitality/ → Project settings
- templates/ → HTML templates
- static/ → CSS, JS, Images
